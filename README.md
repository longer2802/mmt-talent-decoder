# MMT 天賦設計解碼

手機優先的個案天賦設計對照工具。第一版先使用示範資料，並保留 Google Sheets CSV 載入入口。

## 試算表欄位建議

正式串接時，建議把每一位個案整理成一列，欄位名稱可使用：

- 姓名
- 生日
- 風
- 火
- 水
- 土
- 天賦牌
- 導師牌
- 陰影牌
- 流年牌
- 生命靈數上
- 生命靈數左
- 生命靈數右
- 備註

多張牌可以用逗號或頓號分隔，例如 `3、4、5、11、12`。

## 使用方式

直接開啟 `index.html` 即可使用。若要載入 Google Sheets，請先把試算表發佈成 CSV，或使用可匯出 CSV 的分享連結。
